Exemplos de Operadores

Operadores são usados para realizar operações em variáveis e valores.

            
                // Operadores aritméticos
                let soma = 5 + 3;
                let subtracao = 5 - 3;
                let multiplicacao = 5 * 3;
                let divisao = 5 / 3;

                // Operadores de comparação
                let igual = (5 == 5);
                let diferente = (5 != 3);
