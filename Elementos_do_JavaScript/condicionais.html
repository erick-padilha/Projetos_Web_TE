Exemplos de Condicionais

Condicionais são usadas para executar diferentes ações com base em diferentes condições.

            
                // Exemplo de condicional if...else
                let hora = 10;

                if (hora < 12) {
                    console.log("Bom dia!");
                } else {
                    console.log("Boa tarde!");
                }
