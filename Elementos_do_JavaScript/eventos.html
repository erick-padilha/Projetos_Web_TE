Exemplos de Eventos

Eventos permitem que o JavaScript responda a interações do usuário ou a mudanças no documento.

            
                // Adicionando um evento de carregamento do DOM
                document.addEventListener("DOMContentLoaded", function() {
                    console.log("O DOM foi completamente carregado");
                });
