Exemplos de Eventos de Tempo

Eventos de tempo permitem executar código em intervalos específicos ou após um tempo definido.

            
                // setTimeout: executa uma função após um tempo definido (milissegundos)
                setTimeout(function() {
                    console.log("Isso será exibido após 2 segundos");
                }, 2000);

                // setInterval: executa uma função repetidamente em intervalos de tempo definidos (milissegundos)
                let contador = 0;
                let intervalo = setInterval(function() {
                    contador++;
                    console.log("Contador: " + contador);
                    if (contador >= 5) {
                        clearInterval(intervalo); // Para o intervalo após 5 execuções
                    }
                }, 1000);
